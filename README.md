# CS:GO Match Outcome Prediction & Analytics 🎮📊

## 📌 Project Overview
Developed a predictive modeling and data analytics pipeline to analyze Counter-Strike: Global Offensive (CS:GO) match metrics. By investigating round dynamics, player statistics, and weapon economies, this project uses machine learning to predict match outcomes and uncover strategic indicators that drive competitive wins.

## 🛠️ Tech Stack & Libraries
- *Language:* Python
- *Environment:* Jupyter Notebook
- *Core Libraries:* - *Data Engineering:* Pandas, NumPy
  - *Visualization:* Matplotlib, Seaborn
  - *Machine Learning:* Scikit-Learn

## 🚀 Key Analysis Workflow
- *Data Preprocessing & Cleaning:* Cleansed competitive match history datasets, handled missing values, and encoded categorical match factors (like map names and team sides).
- *Feature Engineering:* Extracted relevant gameplay indicators (weapon tiers, health pools, or economy status) to isolate strong predictive variables.
- *Exploratory Data Analysis (EDA):* Visualized win rates across different maps, round-by-round momentum, and weapon usage correlations using Seaborn distribution plots.
- *Predictive Modeling:* Trained and optimized classification models (such as Logistic Regression / Ensemble methods) via Scikit-Learn to forecast winning teams based on mid-match states.

## 📊 How to Run the Project
1. Clone this repository.
2. Install the necessary packages: pip install pandas numpy matplotlib seaborn scikit-learn
3. Launch Jupyter Notebook, open CSGO, and run the cells sequentially.
